EJECUCIÓN DEL CÓDIGO:
Dirigirse a la carpeta ./src, abrir una terminal desde esa ubicación y ejecutar el comando:
>>> python main.py

Se necesita tener instaladas las librerías numpy, sympy y matplotlib
Si no se tienen instaladas, ejecutar los siguientes comandos desde la terminal:
>>> pip install numpy
>>> pip install sympy
>>> pip install matplotlib
